[Diogo Guimaraes]

Este trabalho apresenta o desenvolvimento de um sistema de irrigacao com integracao IOT, como requisito para avaliação do Projeto Integrador IV do curso de Tecnologia em Desenvolvimento de Software Multiplataforma da Fatec Franca. O projeto visa dar autonomia e tranquilidade aos donos de jardins e plantas quando estao longe de casa a lazer ou a trabalho, podendo acompanhar como esta a irrigacao de suas plantas mesmo longe de casa.

---

## INTRODUÇÃO

### Contextualização

Jardins e plantas que acabam abandonados devido a rotina ou viagens a trabalho e lazer.

### Problemática

Falta de cuidado com a irrigacao das plantas.

### Objetivos

Sanar o problema apresentado, garantindo autonomia e tranquilidade aos donos de jardins e plantas.

## INSTALAÇÃO E CONFIGURAÇÃO

### Pré-requisitos

Certifique-se de ter instalado:
- Node.js (versão 18.x ou superior)
- npm ou yarn
- [SGBD escolhido] (versão X.x ou superior)
- Git

### Clonagem do Repositório

```bash
git clone https://github.com/FatecFranca/DSM-G01-PI4-2025-2.git
cd DSM-G01-PI4-2025-2
```

### Configuração do Backend

```bash
cd backend
npm install
```

Inicie o servidor:

```bash
npm run dev
```

Inicie a aplicação:

```bash
npm start
```

A aplicação estará disponível em `http://localhost:3000`
